Raft.NET
=========
A .Net port of the RAFT consensus algorithm.

WORK IN PROGRESS

Technologies used:
 - LMAX Disruptor (.Net port)
 - Protocol Buffers
 - Stateless
 - WCF
 - LightInject
 - Unbuffered IO
